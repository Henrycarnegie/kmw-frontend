# 🌍 KnowMyWorld – NGO Digital Platform

## Overview
KnowMyWorld is a comprehensive NGO web platform designed to connect individuals, learners, and stakeholders through education, community engagement, and social impact initiatives.

The platform integrates features of a Learning Management System (LMS), membership services, and community forums to support global awareness and knowledge sharing.

---

##  Objectives
- Promote global awareness and education  
- Provide accessible online learning opportunities  
- Enable community engagement through forums and webinars  
- Support NGO activities through donations and memberships  

---

## Key Features

###  User Management
- User registration and authentication  
- Role-based access (Admin, Moderator, Member, Instructor, Stakeholder)  
- Profile management  

### Membership & Payments
- Paid membership subscriptions  
- Secure online payment integration  
- Access to exclusive member benefits  

###  Learning Management System (LMS)
- Course enrollment and management  
- Progress tracking for learners  
- Downloadable course materials  
- Course completion tracking  
- Certificate generation upon completion  

### Courses & Content
- Browse and purchase courses  
- Offer and manage courses (for instructors)  
- Structured learning modules  

### Webinars & Events
- Live and recorded webinars  
- Event registration system  
- Notifications and reminders  

### Donations
- One-time and recurring donations  
- Transparent donation tracking  

### Community Forum
- Discussion forums for members  
- Moderation system  
- Topic-based conversations  

### Stakeholders & Team
- Dedicated stakeholder section  
- About Us page  
- Team member profiles  

---

## Tech Stack

### Frontend
- Drupal (Content management and UI)

### Backend
- Python (APIs and business logic)

### Development Methodology
- Agile (Sprint-based iterative development)

---

## System Architecture
- Drupal handles frontend rendering and content management  
- Python backend provides APIs and application logic  
- Database stores user data, course progress, transactions, and forum activity  

---

## Security Features
- Secure authentication and authorization  
- Encrypted payment processing  
- Data privacy protection

##  Installation & Setup

### Prerequisites
- Drupal installed  
- Python 3.x  
- MySQL or PostgreSQL  

### Setup Steps
1. Clone the repository  
2. Set up the Drupal frontend  
3. Configure Python backend services  
4. Connect the database  
5. Run the development server
